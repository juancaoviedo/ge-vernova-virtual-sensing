# Quick Task 260620-6s5 — AGMS vs DERMS (from original patent text)

## Objective

Answer two questions **grounded in the original patent text** (the six image-only PDFs in
`docs/`), not from prior summaries/memory:

1. **AGMS vs a traditional DERMS** — where they are the same, where they complement each
   other, where they differ, where one extends the other.
2. **Definitive answer:** is AGMS only for the distribution system, or also transmission?

## Method

- The six `docs/patent-*.pdf` are **image-only** (0 text layer). OCR each with `ocrmypdf
  --force-ocr --sidecar` into `260620-6s5/ocr/<patent>.txt` (originals untouched; output
  PDFs discarded to /tmp).
- Grep the OCR text for the target vocabulary and pull original passages **with page refs**:
  - DERMS / DER / "distributed energy resource(s)" / inverter / "reactive power" /
    "Volt/VAR" / microgrid / "demand response" / aggregat*
  - Scope language: "field of the invention", background, "transmission", "distribution",
    "transmission and/or distribution", voltage levels, "grid edge", "power grid".
- Map page numbers from the OCR sidecar (form-feed `\f` separated) back to PDF pages.

## Tasks

1. **OCR + index** — produce the six sidecar text files; build a grep index of hits for the
   target vocabulary with page numbers.
2. **EXTRACTION artifact** — `260620-6s5-EXTRACTION.md`: original quotes (verbatim, OCR), each
   tagged `[patent · p.N]`, grouped by theme (scope/applicability; DER & control; architecture
   vs management).
3. **Analysis** — `260620-6s5-ANALYSIS.md`: AGMS-vs-DERMS comparison (same / complementary /
   different / extends) + the distribution-vs-transmission verdict, every claim cited to an
   extracted quote.

## Verify / Done

- Six OCR sidecars exist and are non-empty.
- EXTRACTION.md quotes are verbatim from OCR with patent + page refs.
- ANALYSIS.md gives a clear AGMS-vs-DERMS comparison and a definitive distribution-only
  verdict, each grounded in cited original text (no memory-only claims).

A new **patents-appendix study doc** delivered as a Markdown + self-contained HTML pair,
matching the existing AGMS patent-notes pattern, and wired into the study-site build.

**Content** — an exhaustive, ORACS-framed inventory of distribution-level
state-information sources for virtual sensing / DSSE:

- **14-tier source inventory** (substation, feeder/line, phasor/µPMU, AMI, DER,
  topology/model, pseudo-measurements, control-center systems, environmental,
  protection/event, asset-condition, comms substrate, protocols, non-utility) — every
  row tagged "what it observes" + "how you reach it (protocol)".

  weight/covariance treatment.
- **Classification 3** — deployment placement (substation-edge / field FAD / ADMS-cloud)
  by ORACS reachability, including a source→placement reachability matrix.
- **Classification 4** — feeder archetypes (rural radial / suburban DER-heavy / urban
  networked) and which sources each realistically has.
- **Synthesis** tying the four classifications into the spine of the deployment proposal.

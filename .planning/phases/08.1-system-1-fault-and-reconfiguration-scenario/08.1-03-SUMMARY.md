## Decisions Made

- uid `ieee33-fault-v1` chosen explicitly to be distinct from `ieee33-state-v1`; PATTERNS.md notes duplicate UIDs cause silent dashboard replacement in Grafana
- All-bus voltage envelope (panel 1, no bus_id filter) chosen for SPEC-8 to show collapse vs healthy buses; dead-zone-specific panel (panel 5, bus_id 8-17 filter) added as D-13 extra for detail
- Restored min-voltage stat (panel 7) scoped to `range(start: 18:01Z, stop: 18:03Z)` + `energised=="1"` to prevent the ISO-block zero-fills from dragging the aggregate min to 0
- Topology/event table (panel 8) uses Flux `pivot` to present faulted_line_id, tie_closed, tie_id, n_dead_buses, dead_buses as columns per timestamp row — readable ground-truth log for System 2 integration

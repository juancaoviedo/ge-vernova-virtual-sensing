## Rehearsal Protocol

*One page. Read this before your first session. Follow it every session.*

### Method: Active Recall + Spaced Repetition

**The core discipline (do this on every tracker row):**

1. Read the **Prompt** column only. Cover (or don't look at) the answer key.
2. Say the answer **aloud**. Not in your head — out loud. Record yourself if possible.
3. Time yourself. Check the Duration against the target in the table below.
4. **Only then** check the answer in the source document.
5. Mark **Confidence**:
   - **✗** = Did not get it / blank / completely wrong
   - **⚠** = Partial — got the opening but missed key bullets or ran over
   - **✓** = Solid — hit the key points within the target time
6. Any row marked ✗ or ⚠ goes on your **next-session repeat list** (see "Flag the 2–3 Weakest" below).

**Why out loud?** Retrieval from memory under performance conditions (speaking aloud, being watched, time pressure) is a different skill from reading comprehension. You are training the right skill.

**Why timed?** Every answer has a target. Exceeding it by 50%+ is a partial (⚠), not a pass. The screener or interviewer will move on — you need to close the loop inside the window.

    - gaussian: additive white, zero mean, std=true_sigma
    - gaussian_outliers: with rng.random()<0.03, returns true_val + sign*15*true_sigma; else gaussian.
      Over a full day run the realized outlier fraction is ≈3% (statistically distinguishable; heavier tails)
    - instrument: per-sensor fixed bias + AR(1) correlation + quantization; mean error is nonzero
      (bias), step-to-step noise is autocorrelated (ρ≈0.7), values land on the LSB grid
    - multirate_async: a class with cadence k emits only at step_idx % k == 0; snapshot ignores cadence.
      For source=day AMI (k=4) appears at 96/4=24 steps; for source=fault SCADA (k=2) at 20 steps,
      AMI (k=10) at 4 steps
    - assumed_sigma is true_sigma * cfg["assumed_sigma_scale"] regardless of noise model (never fed
      into noise generation)
  </behavior>

   sqrt(1-rho**2))` and updates prev. Add `_instrument_bias(sensor_key, seed)` returning a
   seed-derived fixed bias via `np.random.default_rng(hash((sensor_key, seed)) & 0xFFFFFFFF).normal(
   0, mc.INSTRUMENT_BIAS_SCALE)`.
2. Add `apply_noise(noise_model, cls, bus, quantity, true_val, true_sigma, rng, instr_state, seed)`:

     if rng.random() < mc.OUTLIER_FRACTION: noise = rng.choice([-1.0,1.0]) * mc.OUTLIER_SPIKE_MULT *
     true_sigma; return true_val + noise`

   same order for a given (sorted) sensor iteration (Pitfall 5).

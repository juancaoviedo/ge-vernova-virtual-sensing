<objective>
Build the AC measurement-model core: a pure-compute module that (1) builds `Ybus` deterministically from a streamed `netmodel/current` topology message applied to the STATIC line/bus impedances of `build_enhanced_33bus()`, verified < 1e-9 against the pandapower reference; (2) implements `h(x)` rows per measurement class (AC injection equations + μPMU identity pickoffs); (3) implements the analytic Jacobian `H`; (4) exposes the FASE sensitivity `S = ∂x/∂p` source; and (5) ships a startup verification gate (Ybus equality, h(x_true) pickoff, FD Jacobian check).

Purpose: This is the physics heart of System 2 — the power flow lives INSIDE `h(x)` (one coupled network-wide estimator, SPEC R3/D2). It has no I/O, no MQTT, no oracle access — fully parallelizable with the publisher (Plan 01).
Output: `ac_model.py` with verified Ybus, h(x), H, S, and a self-checking `verify_model()` entry point.
</objective>

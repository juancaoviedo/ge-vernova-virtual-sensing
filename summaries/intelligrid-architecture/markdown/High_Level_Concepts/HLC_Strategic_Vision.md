# Strategic Vision Based on High Level Concepts

The strategic vision for the IntelliGrid Architecture
reflects the ultimate objectives for a distributed computing infrastructure
that can meet business needs, including configuration requirements, quality of
service requirements, security requirements, and data management requirements.
This strategic vision is based on:

* [Abstract Modeling](HLC_Abstract_Modeling.htm)
* [Security Management](HLC_Security.htm)
* [Network and System Management](HLC_Network_Management.htm)
* [Data Management](HLC_Data_Management.htm)
* [Integration and Interoperability](HLC_Interoperability.htm)

These strategic vision high level concepts are described
in the following subsections.

This high level concepts can be implemented through a
[Tactical Approach](HLC_Tactical_Approach.htm), using
[Environments](HLC_Environment_Links.htm) as a guide to assessing the appropriate
[technology standards](HLC_Standards.htm).

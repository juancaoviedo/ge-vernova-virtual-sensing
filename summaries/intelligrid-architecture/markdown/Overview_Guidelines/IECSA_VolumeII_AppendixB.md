# Stakeholders Engaged and Documents Reviewed

This appendix contains a summary of the stakeholder
engagements that were used to gather input to the design of IntelliGrid Architecture. It should be
noted, that the first table shows the “traditional” stakeholder venue where a
meeting with individuals was convened and discussions and presentations were
exchanged. The latter part of this appendix contains a list of the key
stakeholder documents that were consulted as part of the stakeholder engagement
process.

### W3C The Platform for Privacy Preferences 1.1 (P3P1.1) SpecificationW3C Working Draft

27 April 2004

**URL:** http://www.w3.org/TR/2004/WD-P3P11-20040427/

This
is the specification of the Platform for Privacy Preferences 1.1 (P3P 1.1).
This document, along with its normative references, includes all the
specification necessary for the implementation of interoperable P3P 1.1 applications.
P3P 1.1 is based on the P3P 1.0 Recommendation and adds some features using the
P3P 1.0 Extension mechanism. It also contains a new binding mechanism that can
be used to bind policies for XML Applications beyond HTTP transactions.

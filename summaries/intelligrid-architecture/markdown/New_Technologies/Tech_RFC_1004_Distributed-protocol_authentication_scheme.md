### RFC 1004 Distributed-protocol authentication scheme

**URL:** http://www.ietf.org/rfc/rfc1004.txt

The
purpose of this RFC is to focus discussion on authentication problems in the
Internet and possible methods of solution. The proposed solutions in this
document are not intended as standards for the Internet at this time. Rather,
it is hoped that a general consensus will emerge as to the appropriate solution
to authentication problems, leading eventually to the adoption of standards.

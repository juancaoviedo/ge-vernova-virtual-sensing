### RSA PKCS #8 Private-Key Information Syntax Standard

**URL:** ftp://ftp.rsasecurity.com/pub/pkcs/doc/pkcs-8.doc

This
standard describes a syntax for private-key information. Private-key
information includes a private key for some public-key algorithm and a set of
attributes. The standard also describes a syntax for encrypted private keys. A
password-based encryption algorithm (e.g. one of those described in PKCS #5)
could be used to encrypt private-key information.

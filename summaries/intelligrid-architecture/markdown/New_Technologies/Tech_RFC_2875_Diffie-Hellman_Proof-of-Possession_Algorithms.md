### RFC 2875 Diffie-Hellman Proof-of-Possession Algorithms

**URL:** http://www.ietf.org/rfc/rfc2875.txt

This
document describes two methods for producing an integrity check value from a
Diffie-Hellman key pair. This behavior is needed for such operations as
creating the signature of a PKCS #10 certification request. These algorithms
are designed to provide a proof-of- possession rather than general purpose
signing.

This
part of ISO/IEC 15408 defines the assurance requirements of the standard. It
includes the evaluation assurance levels (EALs) that define a scale for
measuring assurance, the individual assurance components from which the
assurance levels are composed, and the criteria for evaluation of PPs and STs.

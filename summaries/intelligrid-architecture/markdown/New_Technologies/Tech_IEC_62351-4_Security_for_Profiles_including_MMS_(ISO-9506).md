### IEC 62351-4 Security for Profiles including MMS (ISO-9506)

**URL:**  

IEC
62351-4 is a developing standard that specifies how to secure the ISO-9506
(MMS) protocol. It references IEC 62351-3 and adds application level
authentication ability. It has applicability in securing IEC 60870-6 TASE.2
(ICCP) and IEC 61850.

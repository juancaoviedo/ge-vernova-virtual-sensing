### XML Metadata Interchange (XMI)

**URL:**http://www.omg.org/xmi/

XMI defines an XML-based interchange format for UML
metamodels and models (since a metamodel is just a special case of a model), by standardizing XML document formats and DTDs. In so
doing, it also defines a mapping from UML to XML

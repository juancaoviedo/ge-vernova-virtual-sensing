### RFC 2946 Telnet Data Encryption Option

**URL:** http://www.ietf.org/rfc/rfc2946.txt

This
document describes a telnet encryption option as a generic method of providing
data confidentiality services for the telnet data stream. While this document
summarizes currently utilized encryption types and codes, it does not define a
specific encryption algorithm. Separate documents are to be published defining
implementations of this option for each encryption algorithm.

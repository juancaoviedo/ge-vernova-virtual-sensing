### TELNET Protocol

**URL:**http://www.ietf.org/rfc/rfc0854.txt

The TELNET Protocol provides a fairly general,
bi-directional, eight-bit byte oriented communications facility. Its primary
goal is to allow a standard method of interfacing terminal devices and terminal-oriented
processes to each other. The protocol may also be used for terminal-terminal
communication ("linking") and process-process communication
(distributed computation). A TELNET connection uses a TCP connection to
transmit data with interspersed TELNET control information.

**Keywords:** Internet, reliable, terminal access.

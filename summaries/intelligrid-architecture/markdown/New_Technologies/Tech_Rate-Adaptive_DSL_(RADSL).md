### Rate-Adaptive DSL (RADSL)

**URL:****http://www.dslforum.org/**

Rate-Adaptive DSL
(RADSL) is an ADSL technology from Westell in which software is able to
determine the rate at which signals can be transmitted on a given customer
phone line and adjust the delivery rate accordingly. Westell's FlexCap2 system
uses RADSL to deliver from 640 Kbps to 2.2 Mbps downstream and from 272 Kbps to
1.088 Mbps upstream over an existing line.

**Keywords:** Modem, serial, high speed, physical layer, Access Technology,
adaptive

### ISO/IEC 10164-8:1993 Security Audit Trail Function - Information technology - Open Systems Interconnection - Systems Management

**URL:**http://www.iso.ch

Establishes user
requirements for the service definition needed to support the security audit
trail reporting function, defines the service provided by the security audit
trail reporting function, specifies the protocol that is necessary in order to
provide the service, defines the relationship between the service and
management notifications, defines relationships with other systems management
functions, specifies conformance requirements.

**Keywords:** Security, Audit, Non-Repudiation

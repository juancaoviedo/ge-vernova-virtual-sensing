**Fluffy Search** is a search engine for web sites. It can run on any web
server which supports scripting with perl -- no custom server software is needed
-- and can quickly search even large sites with minimal server load.

Its main features are:

**Word highlighting**: When the user follows a link from the search results, the relevant words in the page are highlighted. **Speed**: An index of the contents of your site is generated and used for fast searching. **Controllable**: Fluffy Search can be configured to your requirements. You can even mark areas, such as navigation, in your HTML pages for exclusion from the index. **Frames aware**: If you use multiple framesets on your site, Fluffy Search can link the found pages into the correct frameset.

Fluffy Search is available as a **free** [download](http://www.fluffy.co.uk/fs/download.html), but is
unsupported. There is [documentation](http://www.fluffy.co.uk/fs/docs.html) on this site to
help you configure it to meet your requirements, and some [hints and tips](http://www.fluffy.co.uk/fs/hints.html) from users on
how to get the most from it.

However, Fluffy Clouds can change any of its features for you, from how it
ranks results to searching other sources of information, such as product
databases, and including those results in the found pages. [Email Fluffy Clouds](mailto:solutions@fluffy.co.uk) for a quote for
creating a custom Fluffy Search for your web site.

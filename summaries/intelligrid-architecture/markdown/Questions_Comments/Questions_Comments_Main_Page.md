# Questions and Comments

## 

## Editorial Comments on the IntelliGrid Web Site

Editorial comments are those that will help us make sure the Web
Pages are correct. They include misspellings, broken links, unreadable
text, etc. These comments will be reviewed and corrections implemented
regularly. Please send editorial comments to the following e-mail
address:
[Editorial Comments on the IntelliGrid Web Site](mailto:fcleve@ix.netcom.com?subject=Editorial Comments on IntelliGrid Web Site)

## Issue Comments on the IntelliGrid Architecture

Issue comments are those about issues related to the IntelliGrid
Architecture. These will be reviewed, and may be posted on the
IntelliGrid Web Site as a means to further discussion on these issues.
Please send issue comments to the following e-mail address:
[Issue Comments on the IntelliGrid Architecture](mailto:fcleve@ix.netcom.com?subject=Issue Comment on the IntelliGrid Architecture)

## Help Desk Questions

Simple questions, such as "Where can I find the information on
security on the IntelliGrid Web Site",  can be handled through
e-mail. More complex questions may require follow-up that involves
additional effort. The IntelliGrid Team will address these questions
as thoroughly and as quickly as possible, but may require interactions
with the questioner to determine the scope of the effort. Please send questions to the following e-mail
address:
[Questions on the IntelliGrid Architecture](mailto:fcleve@ix.netcom.com?subject=Questions on the IntelliGrid Architecture)

# *V is for Visual Basic*

**V Series**: Data transmission standards for
telephone lines created by the ITU-T (formerly known as CCITT).
[More...](http://www.itu.int/itudoc/itu-t/rec/v/index.html)

**V.10**: An ITU-T interface recommendation;
electrically similar to RS-423 (unbalanced, high speed).
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v10.html)

**V.11**: An ITU-T interface recommendation;
electrically similar to RS-422.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v11.html)

**V.110**: Async rate adaption (for ISDN) at speeds up
to 19.2 Kbps. [More...](http://www.itu.int/itudoc/itu-t/rec/v/v110.html)

**V.120**: Async rate adaption (for ISDN) at speeds up
to 64 Kbps; uses statistical multiplexing.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v120.html)

**V.17**: An ITU-T standard 2-wire modem for facsimile
applications with rates up to 14 400 bit/s.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v17.html)

**V.21**: An ITU-T standard 300 bps duplex modem
standardized for use in the general switched telephone network; similar to
RS-422 and Bell 103. [More...](http://www.itu.int/itudoc/itu-t/rec/v/v21.html)

**V.22**: An ITU-T standard 1200 bps duplex modem
standardized for use in the general switched telephone network and on
point-to-point 2-wire leased telephone-type circuits; similar to Bell 212.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v22.html)

**V.22bis**: An ITU-T standard 2400 bps duplex modem
using the frequency division technique standardized for use on the general
switched telephone network and on point-to-point 2-wire leased telephone-type
circuits. [More...](http://www.itu.int/itudoc/itu-t/rec/v/v22bis.html)

**V.23**: An ITU-T standard 600/1200-baud modem
standardized for use in the general switched telephone network; similar to Bell
202. [More...](http://www.itu.int/itudoc/itu-t/rec/v/v23.html)

**V.24**: An ITU-T standard interface recommendation
that defines interchange circuits; similar to and operationally compatible with
RS-232. [More info.](http://www.itu.int/itudoc/itu-t/rec/v/v24.html)

**V.25**: An ITU-T standard automatic answering
equipment and general procedures for automatic calling equipment on the general
switched telephone network including procedures for disabling of echo control
devices for both manually and automatically.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v25.html)

**V.25bis**: An ITU-T standard synchronous and
asynchronous automatic dialing procedures on switched networks.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v25bis.html)

**V.26**: An ITU-T standard 2400 bps modem standardized
for use on 4-wire leased telephone-type circuits; similar to Bell 201 B.
[More info.](http://www.itu.int/itudoc/itu-t/rec/v/v26.html)

**V.26bis**: An ITU-T standard 2400/1200 bps modem
standardized for use in the general switched telephone network; similar to Bell
201 C. [More...](http://www.itu.int/itudoc/itu-t/rec/v/v26bis.html)

**V.26te****r**: An ITU-T standard 2400 bps duplex
modem using the echo cancellation technique standardized for use on the general
switched telephone network and on point-to-point 2-wire leased telephone-type
circuits. [More...](http://www.itu.int/itudoc/itu-t/rec/v/v26te.html)

**V.27**: An ITU-T standard 4800 bps modem with manual
equalizer standardized for use on leased telephone-type circuits; similar to
Bell 208A. [More...](http://www.itu.int/itudoc/itu-t/rec/v/v27.html)

**V.27bis**: An ITU-T standard 4800/2400 bps modem
with automatic equalizer standardized for use on leased telephone-type circuits.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v27bis.html)

**V.27ter**: An ITU-T standard 4800/2400 bps modem
standardized for use in the general switched telephone network; similar to Bell
208 B; used for 4800-bps fax communication.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v27ter.html)

**V.28**: An ITU-T interface recommendation that
defines electrical characteristics for the interchange circuits defined by V.24;
similar to and operationally compatible with RS-232.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v28.html)

**V.29**: An ITU-T standard 9600 bps modem standardized
for use on point-to-point 4-wire leased telephone-type circuits; similar to Bell
209; used for 9600-bps fax communication.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v29.html)

**V.32**: An ITU-T family of 2-wire, duplex modems
operating at data signaling rates of up to 9600 bps for use on the general
switched telephone network and on leased telephone-type circuits.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v32.html)

**V.32bis**: An ITU-T standard duplex modem
operating at data signaling rates of up to 14 400 bps for use on the general
switched telephone network and on leased point-to-point 2-wire telephone-type
circuits. [More info.](http://www.itu.int/itudoc/itu-t/rec/v/v32bis.html)

**V.34**: An ITU-T standard modem operating at data
signaling rates of up to 33 600 bps for use on the general switched telephone
network and on leased point-to-point 2-wire telephone-type circuits.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v34.html)

**V.36**: An ITU-T standard modems for synchronous
data transmission using 60-108 kHz group band circuits; replaced V.35.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v36.html)

**V.42**: An ITU-T standard for an error-correction
protocol for modems using an async-to-sync conversion; defines LAP-M protocol.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v42.html)

**V.42bis**: An ITU-T standard for a protocol, based
on V.42 with added data compression.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v42bis.html)

**V.54**: An ITU-T standard describing loop test
devices for modems. [More...](http://www.itu.int/itudoc/itu-t/rec/v/v54.html)

**V.90**: An ITU-T standard digital modem and analogue
modem pair for use on the Public Switched Telephone Network (PSTN) at data
signaling rates of up to 56 000 bps downstream and up to 33 600 bps upstream.
[More...](http://www.itu.int/itudoc/itu-t/rec/v/v90.html)

**V.Fast**: A preliminary version of V.34 operating
at signaling rates of up to 28 800 bps.

**Virtual circuit**: In packet switching, a
network facility that gives the appearance to the user of an actual end-to-end
circuit; a dynamically variable network connection where sequential data packets
may be routed differently during the course of a virtual connection. Virtual
circuits enable transmission facilities to be shared by many users
simultaneously.

**Virtual Private Network (VPN)**:
A VPN (virtual private network) is a way to use a public telecommunication
infrastructure, such as the Internet, to provide remote offices, individual
users, or remote field locations with secure access to their organization's
network. A virtual private network can be contrasted with an expensive system of
owned or leased lines that can only be used by one organization. The goal of a
VPN is to provide the organization with the same capabilities, but at a much
lower cost. A VPN works by using the shared public infrastructure while
maintaining privacy through security procedures and tunneling protocols such as
the Layer Two Tunneling Protocol (L2TP). In effect, the protocols, by encrypting
data at the sending end and decrypting it at the receiving end, send the data
through a "tunnel" that cannot be "entered" by data that is not properly
encrypted. An additional level of security involves encrypting not only the
data, but also the originating and receiving network addresses. VPNs provide
security router-to-router, but do not provide application-to-application
security.

**Voltage Deviation -**
Slow change of the voltage taking place with a rate of less than 1% of
the rated voltage per second.

**Voltage Oscillations-**
Short-term changes of voltage taking place with a rate of more than 1
% of the rated voltage per second.

**Voltage

service of the bulk electric power supply system.  [**[DOE

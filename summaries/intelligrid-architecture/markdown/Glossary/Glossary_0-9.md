# **0-9**

**1000BASE-CX****:** A version of the
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard for
[Gigabit Ethernet](Glossary_G.htm#Gigabit_Ethernet) using shielded
twisted pair cable ([STP](Glossary_S.htm#STP)). The maximum length of
a single segment is 25 meters. [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**1000BASE-LX****:** A version of the
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard for
[Gigabit Ethernet](Glossary_G.htm#Gigabit_Ethernet) using optical
fiber (multi- or single-mode) and long-wave laser. The maximum lengths of a
single segment are 550 meters (multimode) and 5000 m (single mode). [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**1000BASE-SX****:** A
[baseband](Glossary_B.htm#Baseband) signaling version of the
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard for
[Gigabit Ethernet](Glossary_G.htm#Gigabit_Ethernet) using multimode
optical fiber and short-wave laser. The maximum length of a single segment is
550 meters. [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**1000BASE-T****:**A
[baseband](Glossary_B.htm#Baseband) signaling version of the
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard for
[Gigabit Ethernet](Glossary_G.htm#Gigabit_Ethernet) using un-shielded
twisted pair cable ([UTP](Glossary_U.htm#UTP)). The maximum length of
a single segment is 25 meters. [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**100BASE-FX****:** A
[baseband](Glossary_B.htm#Baseband) signaling version of the
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard for
[Fast Ethernet](Glossary_F.htm#Fast_Ethernet) using 2 optical fibers.
[[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**100BASE-T****:** An implementation of the
Ethernet [IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard on
unshielded, twisted-pair wiring ([UTP](Glossary_U.htm#UTP)), using
[baseband](Glossary_B.htm#Baseband) signaling, at 100 Mbps. The
maximum length of a single segment is 100 meters. [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**100BASE-T4****:** A
[baseband](Glossary_B.htm#Baseband) signaling version of the
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard for
[Fast Ethernet](Glossary_F.htm#Fast_Ethernet) using four
[UTP](Glossary_U.htm#UTP) pairs.
[[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**100BASE-TX****:** A
[baseband](Glossary_B.htm#Baseband) signaling version of the
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard for
[Fast Ethernet](Glossary_F.htm#Fast_Ethernet) using 2
[UTP](Glossary_U.htm#UTP) or [STP](Glossary_S.htm#STP)
pairs. [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**100VG-AnyLAN:** A proposed 100-Mbps
[LAN](Glossary_L.htm#LAN) topology over [UTP](Glossary_U.htm#UTP) as suggested by [Hewlett Packard](http://www.hp.com).

**10BASE2**: An implementation of the Ethernet
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard on thin
[RG58](Glossary_R.htm#RG-58_/U:)
[coaxial cable](Glossary_C.htm#Coaxial_Cable), a baseband medium, at
10 Mbps. It is a [bus topology](Glossary_B.htm#Bus_Topology)
[LAN](Glossary_L.htm#LAN) that uses [baseband](Glossary_B.htm#Baseband) signaling with a maximum segment length of 185 meters. Also known
as [Thin Ethernet](Glossary_T.htm#Thin_Ethernet) or Thinnet.
[[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**10BASE5****:** The original Ethernet
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard as defined by DEC(tm),
Xerox(R), and Intel(R), which uses thick
coaxial cable. It is a [bus topology](Glossary_B.htm#Bus_Topology)
[LAN](Glossary_L.htm#LAN) that uses [baseband](Glossary_B.htm#Baseband) signaling with a maximum segment length of 500 meters. Also known
as [Thick Ethernet](Glossary_T.htm#Thick_Ethernet) or Thicknet.
[[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**10BASE-F**: The 10BaseF standard developed by the
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) committee defines the use of
fiber for Ethernet. **10BaseFB** allows up to 2km per segment (on multi-mode
fiber) and is designed for backbone applications such as cascading repeaters. **10BaseFL** describes the standards for the fiber optic links between stations
and repeaters, again allowing up to 2 km per segment on multi-mode fiber. In
addition, there are the **10BaseFP** (Passive components) standard and the
FOIRL (Fiber Optic Inter-Repeater Link), which provides the specification for a
fiber optic MAU (Media Attachment Unit) and other interconnecting components.
[[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**10BASE-T****:** An implementation of the
Ethernet [IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard on
unshielded, twisted-pair wiring, a baseband medium, at 10 Mbps. The maximum
length of a single segment is 100 meters. [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**10BROAD36****:** An implementation of the
Ethernet [IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard using
analog PSK modulation. [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**1BASE5****:** An implementation of the Ethernet
[IEEE 802.3](Glossary_I.htm#IEEE_802.3) standard for low data-rate.
Uses twisted-pair cable and daisy-chained connection with a range of 185 meters
at 1 Mbps. In AT&T ([NCR](http://www.ncr.com))’s implementation
called [StarLAN](http://www.ncr.com/index1.htm). [[IEEE 802.3](Glossary_I.htm#IEEE_802.3)]

**23B+D****:** [ISDN](Glossary_I.htm#ISDN)
primary-rate service consisting of 23\* 64 Kbps
[B Channel](Glossary_B.htm#B_Channel)s and one 64 Kbps
[D Channel](Glossary_D.htm#D Channel). Overhead is 8 kbps which adds
up to 1.544 Mbps (same as [DS-1](Glossary_D.htm#DS-1)/[T-1
Line](Glossary_T.htm#T-1_Line)). See also [PRI](Glossary_P.htm#PRI (Primary Rate Interface)).
[[ISDN](Glossary_I.htm#ISDN)]

**2B+D****:** [ISDN](Glossary_I.htm#ISDN)
basic-rate service consisting of two 64 Kbps [B Channel](Glossary_B.htm#B_Channel)s and one 16 Kbps [D Channel](Glossary_D.htm#D Channel).
Overhead is 48 kbps which adds up to 192 kbps. See also
[BRI](Glossary_B.htm#BRI (Basic Rate Interface)). [[ISDN](Glossary_I.htm#ISDN)]

**3DES**:
[DES](Glossary_D.htm#Data_Encryption_Standard) run 3 times. 
[[WhatIs.com](http://whatis.techtarget.com/)]

**56K leased line****:** Permanent
network connection leased from a telecommunication provider. The
leased line uses special high-speed phone lines and sends/receives data at 56
Kbps.

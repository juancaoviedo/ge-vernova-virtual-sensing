## Narrative

### Pre-arming of Remedial Action Schemes (RAS)

This
application receives pre-arming signals from an upper level of control
and changes the settings (tuning parameters) of distribution-side
remedial action schemes (RAS), e.g., load-shedding schemes (a
component of self-healing grid) or intentional DER islanding.
